・Filter-empty (絞り込みの結果 0 件) ≠ truly-empty (record nào cũng chưa có). Copy + CTA khác nhau hoàn toàn. ・Loading dùng skeleton row giữ đúng width column — không spinner full-page (gây jump khi data về). ・Error phải kèm code + req-id để user copy gửi support — không chỉ "Something went wrong". ・No-permission (chưa render trong file này): tương tự error state nhưng icon là khóa, CTA = "管理者へ依頼".
System rules — recap:
・1 shell .tbl-shell với 1px border + 6px radius. Không shadow. Tất cả region chỉ là border-bottom / border-top.
・Order region chuẩn: views · toolbar · filter-bar · bulk (replaces toolbar khi có selection) · table · footer · pagination.
・Mọi numeric đều font-variant-numeric:tabular-nums + text-align:right. Header dấu sort cũng phải align cùng cell content.
・Status dùng chip-* đã có sẵn — không bao giờ tạo màu cho status mới. Prefer 朱 attention over 茜 destructive với non-destructive alerts (遅刻 · 早退).
・Row states: default · hover · selected · is-new · is-error · is-editing · disabled · expanded · group-row. Mỗi state chỉ thay background bằng color-mix, không thay padding / height.
・Confirmed records (確定済) là frozen — không double-click edit, không bulk-action destructive.
・Pagination chọn theo dữ liệu: numbered cho list, load-more cho feed, cursor cho time-series.
・Import / Export luôn có stepper preview — không apply trực tiếp; default UTF-8 + BOM, có SHIFT-JIS option.
