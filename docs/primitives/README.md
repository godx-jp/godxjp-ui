# Primitives

Raw UI primitives grouped by Ant Design category. Each demo is a standalone `Demo()` component with default token output.

Primitive examples are intentionally strict:

- Use only exported UI components and their props.
- Do not pass story-level `className` or inline `style`.
- Show default density, spacing, radius, color, state and interaction from centralized tokens.
- Keep product scenarios realistic enough to judge production composition.

Coverage follows the public component barrels under `src/components/*/index.ts`.
