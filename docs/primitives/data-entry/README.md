# Data Entry

Form primitives with default token spacing, density, validation rhythm and picker surfaces.

Included primitive pages: Input, Label, Textarea, SearchInput, Select, Checkbox, CheckboxGroup, Radio, RadioGroup, Switch, Slider, FormField, DatePicker, DateRangePicker, Calendar, TimePicker, ColorPicker, Autocomplete, Upload, Cascader, TreeSelect, Transfer and Command.

The examples use controlled state only where the component requires it. Visual styling remains owned by the primitive.
