# Data Display

Read-only display primitives: cards, badges, tables, state surfaces, structured details, progress, timeline and disclosure overlays.

Included primitive pages: Badge, StatusBadge, CodeBadge, Card, CardStat, Table, DataTable, EmptyState, KeyValueGrid, ProgressMeter, TreeList, Timeline, ScanPanel, Popover, ScrollArea and Collapsible.

Token ownership: spacing, header bands, table density, badge shape and empty-state rhythm come from primitive tokens and base CSS, not story overrides.
