Avatar — Primitive hiển thị dữ liệu — ảnh đại diện và fallback.

```tsx
import { Avatar, AvatarFallback, AvatarImage } from "@godxjp/ui/data-display";

<Avatar>
  <AvatarImage src="/user.png" alt="User" />
  <AvatarFallback>UI</AvatarFallback>
</Avatar>
```
