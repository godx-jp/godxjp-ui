# Tutorials

Tutorials are learning-oriented. They guide you through completing a concrete task
from beginning to end so that you understand how the pieces fit together.
Follow each tutorial in order; later tutorials build on the skills from earlier ones.

| Tutorial | What you build | Time |
|---|---|---|
| [01 Getting started](./01-getting-started.md) | A brand-compliant page using `@godxjp/ui` | 15 min |
| [02 Theming](./02-theming.md) | Dark mode + custom accent via a tenant override | 20 min |
| [03 Shell composition](./03-shell-composition.md) | Full portal shell: AppShell + Sidebar + Topbar + CommandPalette | 30 min |
| [04 Add a new primitive](./04-add-a-new-primitive.md) | Contribute a `ProgressBar` primitive upstream | 30 min |

**Prerequisites for all tutorials:** Node 22+, pnpm 9+, a React 19 project using Vite 8+.
